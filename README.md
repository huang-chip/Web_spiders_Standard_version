## Git操作的坑
每次项目发生改动前:
```Git
git pull
# 这样可以将远程仓库及时同步到本地
```
每次提交项目标准操作  
```Git
git add .
git commit -m "写一些备注"
git push 
# 这样可以将远程仓库及时同步到本地
```
